# Online Voting System

A secure web-based voting application developed using Python and Oracle SQL. The system allows authenticated users to cast votes securely while preventing duplicate voting. It includes role-based access control for administrators and voters, ensuring transparency and data integrity throughout the election process.

## Features
- User Authentication
- Role-Based Access Control
- Secure Vote Casting
- Duplicate Vote Prevention
- Result Management
- Oracle SQL Database Integration

## Technologies Used
- Python
- Oracle SQL
- HTML
- CSS
